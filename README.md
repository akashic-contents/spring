# spring

赤い箱をドラッグすると、ドラッグした座標に追従して赤い箱が動きます。
ドラッグが解除されたとき、事前のドラッグ操作に合わせて箱を投げることができます。
オレンジ色のオブジェクトに箱を押し付けることで、バネのような動作で押し付けた方向とは反対に箱を加速させることができます。

## 実行方法

以下のコマンドで実行できます。

```
npm install
npm run build
npm start
```
